# Deepak Bishnoi — Portfolio

3D portfolio website built with **React**, **TypeScript**, **Tailwind CSS**, **Framer Motion**, and **Three.js**.

## Sections

- Hero — Full-screen intro with magnetic portrait effect
- Marquee — Scroll-driven GIF marquee rows
- About — Character-by-character reveal animation
- Services — n8n, chatbots, web dev, WhatsApp, AI integrations
- Skills — Animated proficiency bars
- Projects — 3 sticky-stacking project cards
- Simulator — Interactive n8n pipeline mockup
- Contact — Inquiry form + Fiverr booking

## Deploy

```bash
npm install
npm run dev    # local dev server
npm run build  # production build
```

Auto-deploys to Vercel — just push to GitHub and import.
